# Dasar Pemrograman

Mata kuliah ini membahas konsep dasar pemrograman menggunakan bahasa C++,
meliputi struktur program, input/output, operator, percabangan, dan perulangan.

## Tujuan Pembelajaran
Setelah mengikuti mata kuliah ini, diharapkan mahasiswa mampu:
- Memahami struktur dasar program C++
- Menggunakan input dan output standar
- Mengimplementasikan operator aritmatika dan logika
- Menggunakan struktur kontrol seperti if-else dan switch
- Membuat program sederhana untuk menyelesaikan masalah

## Materi yang Dipelajari
- Pengenalan C++
- Variabel dan tipe data
- Operator aritmatika
- Percabangan (if-else, switch)
- Perulangan (for, while, do-while)
- Array dasar

## Struktur Folder
- notes/ : Catatan teori dan rangkuman materi
- praktikum/ : Kumpulan latihan coding
- tugas/ : Tugas terstruktur dari dosen
- mini-project/ : Proyek kecil sebagai penerapan konsep

## Tools yang Digunakan
- Compiler: g++
- Editor: Visual Studio Code

Melalui mata kuliah ini, saya belajar membangun logika dasar pemrograman
